# Contribution Workflow

1. Add or update module lesson files under `docs/lms/modules/<module-id>/`.
2. Register module/lesson entries in `docs/lms/core/registry.js`.
3. Validate navigation and progress in `docs/index.html`.
4. Verify reset behavior and aggregate percentages.
5. Run available repository checks (if none exist, complete manual browser validation).
6. Update governance docs when contract/rules change.
