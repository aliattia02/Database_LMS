# Module Onboarding Quality Gates

Use this checklist for each new module pack onboarding.

- [ ] **Content integrity**: all lesson routes load successfully.
- [ ] **Navigation correctness**: module selector and lesson selector operate without dead links.
- [ ] **Progress persistence**: lesson completion state persists across refresh.
- [ ] **Reset behavior**: global reset clears all checklist progress keys.
- [ ] **Global analytics**: module and LMS percentages update after lesson changes.
- [ ] **UX consistency**: shared shell styling and component behavior remain consistent.
- [ ] **Contract compliance**: module metadata matches required contract fields.
